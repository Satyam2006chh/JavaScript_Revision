Variables in JavaScript -

Variable ek container hota hai jo data store karne ke liye use hota hai.
JavaScript me 3 tarike se variables declare kiye jaate hain:-
var 
let 
const 


1. var-
Purana tarika hai variable declare karne ka.

Characteristics:

Re-declare allowed
Re-assign allowed
Function scoped

var x = 10
var x = 20
x = 30
all allowed



2. let
Modern JavaScript me mostly let use hota hai.

Characteristics:

Re-declare NOT allowed
Re-assign allowed
Block scoped

let a = 10;
a = 5; this is allowed
let a = 30; this will give u the error 


3. const
Value change nahi ho sakti.

Characteristics:

Re-declare NOT allowed
Re-assign NOT allowed
Block scoped

STRINGS  - 
strings are sequences of characters used to represent text. They’re one of the fundamental data types and can include letters, numbers, symbols, and even spaces. You usually wrap them in quotes:

Single quotes: 'Hello
Double quotes: "Hello"
Template literals (backticks): `Hello ${name}`

📌 Immutable: Once created, a string cannot be changed directly. Operations like .toUpperCase() or .replace() return a new string instead of modifying the original.

HIGHER ORDER FUNCTIONS = 

Higher Order Function (HOF) woh function hota hai jo:

1️⃣ Function ko argument ke roop me accept kare
ya 
2️⃣ Function ko return kare
Matlab function ke saath kaam karne wale functions.


function greet(name) {
    console.log("Hello " + name);
}

function processUser(callback) {
    callback("Satyam");
}

processUser(greet);

greet is the call back function 
processuser is the highter order function





REAL EXAMPLES OF THE HIGHER ORDER FUNCTIONS  - 

1️⃣ map() - 
Har element par function apply karta hai aur new array return karta hai.

let arr = [1,2,3]
let result = arr.map((num) =>{
    num * 2;
})
console.log(result); = [2,4,6];


2️⃣ filter()
Condition ke basis par elements select karta hai.

let arr = [1,2,3,4,5]
let result = arr.filter((num) =>{
    num%2==0;
})
console.log(result) = [2,4];


3️⃣ reduce()
Array ko single value me convert karta hai.

let arr = [1,2,3,4];
let sum = arr.reduce((acc,val) => acc + val, 0);
console.log(sum);

where = 
acc → accumulator
val → current value
