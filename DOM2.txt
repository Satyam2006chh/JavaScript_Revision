1. What is an Event in JavaScript = 

An event is a signal that something has happened in the browser,
usually triggered by a user interaction or a browser action, which JavaScript can detect and respond to.

In simpler words:
👉 An event is an action or occurrence that happens in the browser.



Examples:

User clicks a button
User presses a key
Page finishes loading
Form is submitted
Mouse moves

When these actions happen, the browser fires an event.
JavaScript can listen to that event and run code in response.



Someone presses the doorbell → event occurs
The bell rings → notification
You open the door → response

1. What is Event Handling - 

Event handling is the process of executing a specific piece 
of code (function) when an event occurs on an element.

👉 Event happens → JavaScript handles it with a function

===================================

node.event = () => {
   // handle here
}

====================================

<button id = "btn">Click Me !</button>

let button = document.querySelector("#btn")
button.onclick = () => {
    console.log("Button was clicked")
}

What happens here

1️⃣ JavaScript finds the button
2️⃣ It attaches a click event handler
3️⃣ When user clicks → function runs

1️⃣ What is the Event Object ->

The Event Object is a special object automatically 
created by the browser whenever an event occurs, and it contains detailed i
nformation about that event.

In simple words:

👉 When an event happens, the browser creates an object that stores all information about that event.

2️⃣ Example to Understand

User clicks a button.

The browser stores information like:

Which element was clicked
What type of event happened
Mouse position
Time of event
Which key was pressed
All this information is stored inside Event Object.

5️⃣ Important Properties of Event Object = 

e.target  ->   It tells which element triggered the event.
e.type    ->   It tells what type of event occurred.
e.clientX ->   The horizontal position of the mouse pointer when the event occurred
e.clientY ->   The vertical position of the mouse pointer when the event occurred.

1️⃣ What is an Event Listener =

An event listener is a function that waits for a specific event
to occur on an element and executes a callback function when that event happens.


node.addEventListener(event, callback)

<button id="btn">Click Me</button>

let btn = document.querySelector("#btn");

btn.addEventListener("click", function() {
    console.log("Button clicked!");
});

1️⃣ What is removeEventListener() = 

removeEventListener() is a method used to remove a previously attached 
event listener from an element so that the callback function no longer 
executes when the event occurs.

2️⃣ Why Do We Need It?

Sometimes you only want an event temporarily.

Examples:

Disable button after clicking once
Stop a game event
Stop scrolling listener
Stop animation triggers


element.removeEventListener(event, callback)

<button id="btn">Click Me</button>


let btn = document.querySelector("#btn");

function greet(){
    console.log("Hello");
}

btn.addEventListener("click", greet);

btn.removeEventListener("click", greet);

Now when you click the button:
👉 Nothing will happen

1️⃣ What is classList in JavaScript

classList is a property of a DOM element that gives us methods to add, remove, toggle, and check CSS classes of that element.


3️⃣ classList.add()
Definition

classList.add() adds a CSS class to an element.

<div id="box">Hello</div>
.dark{
    background: black;
    color: white;
}

let box = document.querySelector("#box");

box.classList.add("dark");

RESULT = 
<div id="box" class="dark">Hello</div>


4️⃣ classList.remove() = 

classList.remove() removes a CSS class from an element.

box.classList.remove("dark");

RESULT = 
<div id="box">Hello</div>
